Computer Science GCSE

Pre-requisite

Welcome! This revision guide is designed for computer science students. Although this guide is designed for the OCR Computer Science GCSE, more specifically the J276 course, any computer science student can utilize this as a reference guide.

The course is split up into three parts:

  • Component 1 - Computer Systems
  • Component 2 - Computational thinking and algorithms
  • Component 3 - Programming Project

Some content covered in this guide is not covered within the J276 GCSE exam content; content that falls under this umbrella is featured, as I believe it would be beneficial to the reader.

Component 1

Component 1 covers computer systems, security, and the global impact that computer science has in a global context.

  • 1.1 - Systems architecture
  • 1.2 - Memory
  • 1.3 - Storage
  • 1.4 - Wired and wireless networks
  • 1.5 - Network topologies, protocols and layers
  • 1.6 - System security
  • 1.7 - Systems software
  • 1.8 - Ethical, legal, cultural and environmental concerns
